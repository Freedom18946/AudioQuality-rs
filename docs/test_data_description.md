# 测试数据说明 (Test Data Description)

此文件夹包含的音频文件来自专辑《试音发烧人声》，用于功能性测试和演示。

## 专辑信息 (Album Information)

- **专辑名称**: 试音发烧人声
- **艺人**: 群星
- **发行公司**: 香港金百利唱片
- **出版社**: 珠影白天鹅音像出版社
- **版权提供**: 老虎鱼国际唱片有限公司(HK)
- **发行时间**: 2013年07月

## 专辑简介 (Album Introduction)

HI-FI 顶级人声测试天碟，精选十五首发烧人声；被誉为最靓声、最动听的天籁之声。专辑选曲精良，音乐配器及旋律层次分明，极大地突出了人声表现。该专辑是专业音响技师推荐的试机首选碟，也是环球典藏金牌试音大全的最新版本，在专业试音碟中推荐满意度高达95%。
